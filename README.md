# Ревизия второго мозга

Утилита для тех, у кого второй мозг уже переехал на сервер.

Проверяет фактами с диска, что все контуры подключены — память, навыки, правила агента, настройки, хуки, субагенты, подключения и Telegram-бот — достраивает недостающее и ставит ежечасную синхронизацию, чтобы серверная копия не отставала.

## Как запустить

Открой Claude Code в папке своего проекта и вставь:

```text
Прочитай файл по ссылке целиком и дальше веди меня строго по нему:
https://raw.githubusercontent.com/alexandrkuznetsovofficial-web/brain-revision/main/novoselie_revision_audit.md
```

Дальше агент работает сам. Остановит тебя только в трёх случаях: если в перенос попадает личное, если нужно править конфиг бота, если действие необратимо.

Ничего не удаляется: перенос только добавляет файлы, личный контур на сервер не едет.
